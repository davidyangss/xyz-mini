# AI 维护指南

## 工作顺序

1. 先读根目录 [AGENTS.md](../AGENTS.md) 或 [CLAUDE.md](../CLAUDE.md)。
2. 再读本任务涉及目录的 `init.md`。
3. 修改前确认是否涉及私有路径、运行数据、密钥或系统级操作。
4. 修改后同步更新文档索引和双向引用。

## 使用 Warp 等远程终端

remote 主机不需要安装专用 LLM agent。只要 AI 终端工具能读取仓库文档，就可以根据 `AGENTS.md` 和 `init/` 文档链理解环境约束。

这个模式的核心是：

```text
交互入口由终端工具提供，操作语义由项目文档提供。
```

## 文件生成约束

- 不生成真实密钥、真实 token、真实代理地址。
- 不写入真实用户名和个人主机名。
- 不把 `.claude/skills`、`.codex`、私有插件或本机配置作为公开模板内容。
- 示例路径使用 `/xyz/home/<user>`、`/xyz/users/<user>` 或 `<host>` 占位。
